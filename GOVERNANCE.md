This repository is developed and maintained by the **ACTI-X** project.

Contact: [github@acti-x.eu](mailto:github@acti-x.eu)

## Scope

This document governs the **ACTI-X** project and all related repositories.

---

## Roles

### Project Owner

The **ACTI-X** project consortium is the owner and maintainer of this repository.

All final decisions on code inclusion, releases, and project direction
rest with the ACTI-X project.

### Maintainers

Maintainers are granted write access by the ACTI-X project to review
and merge contributions.

A list of maintainers is kept in [MAINTAINERS.md](MAINTAINERS.md).

---

## Decision-Making

All changes are proposed via pull request.

Maintainers review contributions for:

- correctness and code quality
- alignment with project goals and architecture
- test coverage and documentation

Final decisions on inclusion rest with the ACTI-X project.
