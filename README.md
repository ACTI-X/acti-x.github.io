# ACTI-X Repository Template

![ACTI-X](https://github.com/ACTI-X/.github/blob/main/images/acti-x-dc.png?raw=true)

Template repository to bootstrap new ACTI-X projects.

## What This Template Ships

Files that **must** live in every repository (not inherited from the
organization `.github` repo):

- [LICENSE](LICENSE) — Apache 2.0
- [NOTICE.md](NOTICE.md) — ACTI-X copyright, EU funding statement, trademarks
- [GOVERNANCE.md](GOVERNANCE.md)
- [MAINTAINERS.md](MAINTAINERS.md)
- `README.md`

## What's Inherited From `.github`

The following are provided org-wide by
[ACTI-X/.github](https://github.com/ACTI-X/.github) and are inherited
automatically when missing from the project repo — do **not** duplicate
them here unless you need to override:

- `CODE_OF_CONDUCT.md`
- `CONTRIBUTING.md`
- `SECURITY.md`

## Required Per-Project Additions

Every project using this template **must** also add:

- `.gitignore` — tailored to the project's language and tooling.
  For common stacks you can generate one with
  [gitignore.io](https://www.toptal.com/developers/gitignore).
- Extend [NOTICE.md](NOTICE.md) with any project-specific third-party
  attributions (the org-wide copyright and EU funding statement are
  already included).

## Before You Publish — Review Each File

After bootstrapping a new project from this template, **open each file and
fill in / adapt the sections marked below**:

- [README.md](README.md) — replace the entire content with your project's
  README (this file is a template, not the final one).
- [MAINTAINERS.md](MAINTAINERS.md) — fill in the **Active Maintainers**
  table (currently contains `_TBD_` placeholders).
- [NOTICE.md](NOTICE.md) — review the **Third-Party Attributions** section:
  list bundled dependencies and their licenses, or remove the section if
  not applicable.
- [GOVERNANCE.md](GOVERNANCE.md) — review references and confirm they
  match the project's setup; no placeholders by default.
- [LICENSE](LICENSE) — leave pristine (Apache 2.0 canonical text).
